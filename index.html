Apa Itu GitHub?

GitHub adalah platform kolaborasi pengembangan perangkat lunak yang sangat populer.Ini menyediakan berbagai fitur untuk mengelola proyek perangkat lunak,berkolaborasi dengan orang lain,dan melacak perubahan dalam kode sumber.

Langkah-langkah membuat akun digithub:

  • Buka https://github.com dibrowser anda,lalu pilih Sign Up/Daftar
  • Isi Informasi Akun :
    • Nama pengguna(username): Pilih nama pengguna yang unik yang akan digunakan untuk mengidentifikasi anda diGitHub.Pastikan nama pengguna yang anda pilih tersedia.
    • Alamat Email: Masukkan alamat email aktif yang akan digunakan untuk verifikasi dan komunikasi dari GitHub.
    • Kata Sandi(Password):Buat kata sandi yang kuat untuk melindungi akun anda.Pastikan kata sandi anda cukup aman dengan kombinasi huruf besar,huruf kecil,angka,dan karakter khusus.
    • Verifikasi Kata Sandi:Ketik ulang kata sandi yang sama untuk memastikannya.
  • Pilih Rencana gratis:

    • GitHub menawarkan rencana gratis yang hebat untuk pengguna individu. Pilih opsi "Free" untuk menggunakan GitHub secara gratis. Jika Anda ingin mengupgrade akun Anda ke rencana berbayar di kemudian hari, Anda masih dapat melakukannya.

  • Verifikasi Captcha:

    • Ikuti instruksi untuk menyelesaikan verifikasi captcha yang mungkin diminta.

  • Klik"Create Account" atau "Buat Akun":

    • Setelah Anda mengisi semua informasi yang diperlukan dan menyelesaikan verifikasi, klik tombol "Create Account" atau "Buat Akun" untuk membuat akun GitHub Anda.

  • Verifikasi Alamat Email:

    • GitHub akan mengirimkan email verifikasi ke alamat email yang Anda masukkan. Buka email tersebut dan ikuti tautan verifikasi yang diberikan untuk mengaktifkan akun Anda.

    Setelah Anda mengikuti langkah-langkah ini, Anda akan memiliki akun GitHub yang telah dibuat. Anda dapat mulai menggunakan GitHub untuk menyimpan, mengelola, dan berkolaborasi dalam proyek perangkat lunak. Pastikan untuk mengonfigurasi preferensi akun Anda, seperti mengunggah foto profil dan mengatur preferensi keamanan, untuk membuat pengalaman Anda lebih baik.

Cara Upload Project ke Github:

  • Daftar dan buat akun.
  • Buat Repository Baru Setelah masuk ke akun Github,Setelah berhasil masuk ke akun Github Anda, langkah selanjutnya adalah membuat repositori baru untuk proyek Anda. Klik tombol "Repositori Baru" dan lengkapi informasi yang diperlukan seperti nama repositori, deskripsi, dan pilihan privasi. Pilih opsi "Publik" jika Anda ingin membagikan proyek Anda kepada semua orang, atau "Pribadi" jika Anda hanya ingin membagikannya kepada anggota tim tertentu.
  • Setelah repositori baru dibuat, Anda dapat mulai mengunggah proyek Anda.
    1. Buka terminal atau command prompt
    2. Masuk ke folder projectmu dengan perintah "cd [nama folder]"

    3. Inisialisasi Git di folder tersebut dengan perintah "git init"

    4. Ketik perintah "git status" untuk menampilkan informasi tentang status dari repositori Git lokal anda.Perintah ini nuga akan menampilkan daftar file yang telah anda ubah sejak commit terakhir,file yang telah diedit,ditambahkan,dan dihapus.

    5. Tambahkan file-file projectmu ke repository dengan perintah "git add . [nama file]"

      6. Ketik kembali perintah "git status" untuk melihat apakah file berhasil di Upload.Jika berhasil di upload warna dari nama file akan berubah jadi hijau.

    6. Lakukan commit perubahan dengan perintah "git commit -m [pesan commit]"

      7. jika ada pesan seperti gambar diatas,ikuti langkah-langkah seperti dibawah,kita akan diminta untuk config user dan email yang telah kita buat sebelumnya.

      Lalu lakukan commit kembali

    7. Tambahkan URL repository Github sebagai remote repository dengan perintah "git remote add origin [url repository]"

    8. Unggah project anda ke repository dengan perintah "git push -u origin master"

    9. Setelah selesai mengunggah proyek Anda, Anda dapat melihat hasilnya di halaman repositori Github. Semua file dan folder yang telah Anda unggah akan ditampilkan di sana. Selain itu, Anda juga dapat melihat riwayat perubahan dan mengelola file dengan menggunakan fitur-fitur Github seperti pull request, branch, dan lainnya.
